# Superior Trending

Landing page desenvolvida em contexto **profissional/contratual** para a **Superior Trending**, com foco na apresentação institucional e comercial da marca, destacando produtos premium para **petshops** e **veterinárias**.

Este projeto está disponível neste repositório **com autorização para compor portfólio**, como demonstração de solução front-end aplicada a uma demanda real.

## Preview

A aplicação foi publicada para visualização online:

**Demo:** `project-superior-trending.vercel.app`

---

## Sobre o projeto

O projeto foi construído como uma interface moderna, objetiva e visualmente atrativa para fortalecer a presença digital da marca e facilitar a apresentação comercial dos produtos oferecidos.

A proposta da landing page é comunicar de forma clara:

- o posicionamento da empresa no mercado pet
- os diferenciais comerciais
- as marcas trabalhadas
- as condições de compra
- um canal rápido de contato com potenciais clientes

---

## Funcionalidades

- **Hero section** com destaque comercial principal
- **Seção de itens/diferenciais** com apresentação visual dos benefícios
- **Bloco de pedidos** com informações comerciais
- **Carrossel de marcas/parceiros**
- **Rodapé institucional**
- **Scroll suave** para melhorar a experiência de navegação
- **Interface responsiva** para diferentes tamanhos de tela

## Como executar localmente

Clone o repositório:

```bash
git clone https://github.com/ImRobss/project-superior-trending.git
```

Acesse a pasta do projeto:

```bash
cd project-superior-trending
```

Instale as dependências:

```bash
npm install
```

Execute o projeto em ambiente de desenvolvimento:

```bash
npm run dev
```

Para gerar a build de produção:

```bash
npm run build
```

Para visualizar a build localmente:

```bash
npm run preview
```
